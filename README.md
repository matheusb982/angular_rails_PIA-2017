# Projeto de PIA
Projeto desenvolvido para aprender a integração do front-end com Angular e back-end com Ruby on Rails.

Tecnologias utilizadas:

* Ruby 2.4.1
* Rails 5.1.4
* PostgreSQL
* angularjs-rails
